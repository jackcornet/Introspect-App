# Introspect

Introspect is a mindful spending companion designed to help you pause impulse purchases, understand why you want something, and make clearer decisions before you buy.

## Why Introspect
Impulse spending usually happens fast. Introspect slows the moment down so you can choose intentionally, not emotionally.

## What It Does
- Lets you log items before buying
- Adds a cooldown before checkout
- Guides reflection on the reason behind the urge
- Helps you build better buying habits over time
